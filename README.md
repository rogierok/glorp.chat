### glorp.chat

Glorp that is a FULLY browser based, low training time SLM (Small Language Model), a highly advanced chatbot running databaseless.

The entire Model fits in a 24.1 KB (Unzipped and un-minimized) JavaScript file, allowing the user to run this advanced model on any device.
